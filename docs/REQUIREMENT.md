# Employee Leave & Expense Management System

## Roles

1. Admin
2. Manager
3. Employee

---

## Employee

* Login
* View Profile
* Apply Leave
* Cancel Leave
* Submit Expense
* Upload Receipt
* View Leave History
* View Expense History

---

## Manager

* View Team
* Approve Leave
* Reject Leave
* Approve Expense
* Reject Expense

---

## Admin

* Add Employee
* Update Employee
* Delete Employee
* View Dashboards
* Manage All Leaves
* Manage All Expenses
